# Packages

Ce projet utilise une architecture monorepo avec plusieurs packages internes partagés. Ces packages sont situés dans le dossier `packages/` et sont utilisés par les applications.

## Liste des packages

- **[@workspace/ui](/packages/ui)** : Composants d'interface utilisateur basés sur Shadcn UI.
- **[@workspace/form](/packages/form)** : Composants et utilitaires pour la gestion des formulaires.
- **[@workspace/pokeapi](/packages/pokeapi)** : Client typé pour l'API Pokémon.

Ces packages sont gérés via pnpm workspaces et peuvent être importés dans les applications comme n'importe quelle dépendance npm.
